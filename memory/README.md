# Project Memory

Design decisions for **boomoorlog** live here as Markdown files. This folder is the
source of truth for how the game should work. Claude reads these before designing or
building, and updates them automatically as we make decisions.

- [VISION.md](VISION.md) — overall concept, user flow, principles, data dependencies.
- [STATS.md](STATS.md) — per-genus stat block design.

**Convention:** whenever we decide something about how the game works, it gets
recorded in the relevant file here (no need to ask).
