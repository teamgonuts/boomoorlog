# Project Memory

Design decisions for this repo live here as Markdown files. This folder is the source of
truth for how each product should work. Claude reads these before designing or building,
and updates them automatically as we make decisions.

## Active product — Creatures AMS

A single-purpose website that makes Amsterdam locals fall in love with the wildlife on
their own street: a live pixel-art map fed by real open data (Amsterdam trees, iNat,
Waarneming, OSM, KNMI, xeno-canto, …).

- [CREATURES_VISION.md](CREATURES_VISION.md) — concept, audience, principles, user flow.
- [CREATURES_ROADMAP.md](CREATURES_ROADMAP.md) — phased delivery plan (C1–C19).
