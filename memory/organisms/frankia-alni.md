Frankia alni is an actinobacterium that forms nitrogen-fixing root nodules on Alnus (alder) trees, which line many Amsterdam canals and parks. The organism is entirely endosymbiotic within root tissue — invisible above ground but inseparable from the host tree's root zone. Placement at the tree coordinate with tree-rooted is the most accurate representation available in the vocabulary, though the organism technically lives below ground. Movement is none since bacteria are rendered as static sprites. The override from the other default (anywhere / idle-only) pins it correctly to its host tree.
