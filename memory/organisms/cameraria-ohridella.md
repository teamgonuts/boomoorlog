The horse-chestnut leaf miner is a tiny moth whose larvae create pale blotch mines inside the leaves of Aesculus hippocastanum, the horse-chestnut trees that are extremely common in Amsterdam parks and along canals. By late summer infested trees display brown, papery foliage that appears to have turned early; the mines are visible as pale patches across entire canopies. Larvae are fully sedentary miners inside leaf tissue, so tree-canopy / idle-only is appropriate; override from the insect default of flower-visitor as this is a tree-feeding microorganism on Aesculus.
