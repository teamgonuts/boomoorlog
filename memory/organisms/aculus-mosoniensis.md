Aculus mosoniensis is an eriophyid mite that has spread across Europe on tree-of-heaven (Ailanthus altissima), causing leaf curling and witches' broom symptoms. Ailanthus is widespread as a self-sown street and ruderal tree in Amsterdam, so this mite is common wherever the host stands. As with all eriophyids, microscopic and sedentary; tree-canopy + idle-only.
