This entry represents the mixed observation of cavity-nesting tits (Parus) and roosting pipistrelle bats (Pipistrellus) associated with hollow sections of Pterocarya (Caucasian wingnut) trees in Amsterdam. The observation context is the roost or nest cavity rather than active foraging, so idle-only better captures the roosting/resting state than the default tree-flitter. Bird is used as the dominant category since the taxon group is recorded primarily under avian observations; the bat component is a secondary occupant of the same cavities. Tree-canopy placement snaps correctly to the host tree where the cavity is located.
